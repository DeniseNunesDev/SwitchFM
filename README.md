Decidi iniciar, prototipando a ideia do projeto

-> https://www.figma.com/design/Rut56tvDff5e6KyBTkJ76E/SwitchFM?node-id=0-1&t=KKtSVy3BanEAlhyD-1

A partir desse protótipo, pensei "por que não aprimorar, colocando uma API do Spotify para me auxiliar e melhorar a experiência para meu App?" e assim segui com meu planejamento do App
